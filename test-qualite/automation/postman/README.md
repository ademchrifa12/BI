# Pack Postman - Integration API

## Fichier

- `WWI_BI_API.postman_collection.json`

## Variables a definir

- `api_base` : ex `http://localhost:5000/api`
- `jwt_token` : token JWT local obtenu apres login

## Usage

1. Importer la collection dans Postman.
2. Definir les variables d'environnement.
3. Executer les requetes critiques (auth, users, customers, analytics, dashboard).
4. Exporter les resultats/captures vers `preuves/`.
